sendw(To, Msg)
Types:

To = pid() | Name | {Name,Node}
 Name = Node = atom()
Msg = term()

As send/2, but waits for an answer. It is implemented as follows:

sendw(To, Msg) -> To ! {self(),Msg}, receive Reply -> Reply end.

The message returned is not necessarily a reply to the message sent.
